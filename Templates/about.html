About the Historical Consumer Price Index

Here's a brief description of the project: My project is about how the price of food has increased since 1974. From 1974 to 2022, I found data on multiple different categories of food. Some examples are fruit prices, dairy products, meat prices, etc.

Data Cleansing Pipeline

The data cleansing process consists of several steps to clean and format the data for analysis. This converts the cleansed CSV data into a JSON object formatted for generating SVG graphs.

My data cleansing pipeline:

  1. First, I found a set of data on Google that contained data about tons of different types of food.
  2. I then imported my data into Google Sheets and cleansed it of unnecessary information.
  3. Next, I downloaded the Sheet data as a CSV and imported it into VS Code.
  4. Finally, I combined and organized the dataset into a dictionary (by year, and then by category of food)
